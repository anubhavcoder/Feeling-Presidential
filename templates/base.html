About Feeling Presidential

Feeling Presidential is data visualization of common phrases and sentiment analysis in U.S. presidential speeches, analyzing the ten most recent U.S. Presidents, from Kennedy to Obama.

Using machine learning, a Naive Bayes classifier was trained on a hand-tagged set of 70 American political speeches, grouped into positive and negative categories. Sentiment analysis is performed on each Presidents' inaugural speech, their first and last State of the Union speeches, and the most common phrases in each speech.

This was a final project completed in a month for Hackbright Academy, a coding bootcamp for female software engineers. While the data seen here represents the exact output of the classifier, machine learning requires a much higher training set of hundreds (if not thousands) of pre-classified inputs in order to result in higher accuracy. The sentiment analysis presented here is not recommended for use in research, media, or proving your point in an Internet argument.
